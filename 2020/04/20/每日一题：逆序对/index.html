每日一题：逆序对

题意

题意很简单，求长度为n的01串逆序对数量和。( n <= 1e18 )

solution

任意选两个位置 $i, j$ $(i < j)$，令 $a[i] = 1, a[j] = 0$，这样一定能产生逆序对，这样有 $C_n^2$ 种选法。剩下的位置随便放，有$2^{n-2}$种选法，总方案数即为答案。( 注意 1 需要特判 )
